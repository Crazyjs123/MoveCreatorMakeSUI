## Move介绍
Move 是一门专为编写安全、可验证的区块链智能合约而设计的编程语言

## Sui介绍
Sui 是一种高吞吐量、低延迟的区块链，旨在为 Web3 应用程序提供无缝的用户体验. Move语言是Sui区块链开发应用的编程语言
